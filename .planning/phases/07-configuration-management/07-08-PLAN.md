## YAML to TOML Conversion Rules

1. **Nested objects** → Use `[section]` or `[section.subsection]` headers
2. **Arrays of objects** → Use `[[array_name]]` for each item
3. **Inline arrays** → Use `key = ["a", "b", "c"]`
4. **Strings** → Always quote with double quotes
5. **Booleans** → `true`/`false` (lowercase)
6. **Comments** → `#` (same as YAML)

## Example Conversion

**YAML:**
```yaml
server:
  listen: ":8787"
  timeout_seconds: 30
```

**TOML:**
```toml
[server]
listen = ":8787"
timeout_seconds = 30
```
